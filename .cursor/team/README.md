# Cursor AI Developer Team

This folder stores shared communication artifacts for Cursor AI agents.

## Main files

- agent_roster.md: list of agent roles and responsibilities
- communication_protocol.md: how agents communicate
- task_board.md: lightweight task tracking
- integration_map.md: feature/module integration notes
- risk_register.md: known risks
- release_notes.md: release summary

## Folders

- tasks/: task briefs
- handoffs/: agent handoffs
- reviews/: review reports
- decisions/: ADR technical decisions
