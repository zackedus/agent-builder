# Project Brief

Project name:
Agent Team Builder

Project type:
CLI / desktop tooling / library (multi-agent orchestrator)

Main framework:
Python 3.11+, Typer, Flet (generated apps + dashboard), Anthropic + Ollama

Main purpose:
Autonomous multi-agent system that builds Flet desktop applications from natural language prompts

Main users:
Developer (Zaki) building and operating the agent pipeline; end users of generated desktop apps

Core principles:
- Stable
- Maintainable
- Secure
- Modular
- Testable
- Minimal duplication
- Easy to extend
